# FlowMind Frontend

AI Workflow Automation Platform Frontend

## Stack

- React
- TypeScript
- Vite
- TanStack Query
- Zustand
- React Hook Form
- Zod
- Tailwind CSS
- shadcn/ui
- React Flow
- Socket.io Client

## Features

- Authentication
- RBAC
- Organizations
- Workflow Builder
- Workflow Scheduling
- Execution Monitoring
- Realtime Updates
- Knowledge Bases
- RAG
- API Keys
- Audit Logs

## Setup

npm install

npm run dev
